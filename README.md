## 脚本详细说明
### 1. number_bomb.py - 数字炸弹游戏
#### 游戏规则
- 程序随机生成1-100之间的"炸弹数字"
- 玩家每次输入一个猜测数字
- 程序提示猜测数字与炸弹的大小关系
- 猜中炸弹数字游戏结束，显示猜测次数
#### 运行方法
bash
python number_bomb.py
#### 使用示例
数字炸弹游戏开始！
炸弹在1到100之间的某个数字
请输入1-100 之间的整数：50
太大了！炸弹数字比 50 小
当前范围：1 - 49
请输入1-100 之间的整数：25
太小了！炸弹数字比 25 大
当前范围：26 - 49
请输入1-100 之间的整数：37
恭喜你猜中了炸弹数字 37
你总共猜了 3 次
#### 功能特性
智能范围提示，逐步缩小猜测范围
完整的输入验证（整数检查、范围检查）
猜测次数统计
友好的用户交互界面

#### 代码亮点
使用二分法查找思路

### 2. luckydog_draw.py - 幸运抽签程序
#### 程序功能
- 收集迟到学生姓名列表
- 随机抽取2名"幸运儿"回答问题
- 防止空输入和重复输入
- 友好的结果展示
#### 运行方法
bash
python luckydog_draw.py
#### 使用示例
将抽取两个迟到的同学上台回答问题
你迟到了，请输入你的姓名，最后一名同学输入'开始'进行抽签
请输入你的姓名:张三
请输入你的姓名:李四
请输入你的姓名:王五
请输入你的姓名:开始
抽签结果来喽
请回答第一个问题:李四
请回答第二个问题:张三
下次不许再迟到了！！！！
#### 功能特性
收集用户输入
随机不重复抽样
输入验证
结构化结果输出
#### 代码亮点
随机抽样和结果组织

## 常见问题解答
**Q: 运行脚本时出现编码错误？**
A: 确保文件保存为UTF-8编码，或在文件开头添加 `# -*- coding: utf-8 -*-`
**Q: 如何修改数字炸弹的游戏范围？**
A: 修改 `random.randint(1, 100)` 中的参数即可调整范围
**Q: 抽签程序可以抽取更多人吗？**
A: 修改 `random.sample(names, 2)` 中的数字即可调整抽签人数
